# ecgClassifierAPI
ECG classifier api and demo

Installation:

```bash
git clone https://github.com/sushant-thapa/ECGAnalysisUsingDecisionTree
cd ecgclassifierinterface

./init.sh
./mergeModel.sh
```

Starting the server:
```bash
./start.sh
```


Remember to split the model before commiting:
```bash
./splitModel.sh
```
